metayaml.cli.filter(query, directory, abs_path)¶

Create rclone filter rules for files and directories matching a specific attribute stored in YAML meta data sidecar files.

 Examples: metayaml filter “score > 5” metayaml filter “djohn in users” metayaml filter “is_example = True” metayaml find “description = ‘foo’ “

metayaml.cli.find(query, directory, abs_path)¶

Find files matching a specific attribute stored in YAML meta data sidecar files.

 Examples: metayaml find “score > 5” metayaml find “djohn in users” metayaml find “is_example = True” metayaml find “description = ‘foo’ “

metayaml.cli.export(root_dir)¶

Export all YAML meta data to one YAML file

 root_dir should be / to ensure that all meta data files are collected. Otherwise, recursive inheritance of attributes can not be retrieved from the exported data
